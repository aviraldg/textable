Textable

A basic library for creating pure-text tables written in Kotlin.

To install, add:

   repositories {
        jcenter()
        maven { url "https://jitpack.io" }
   }
   dependencies {
         compile 'com.aviraldg:textable:master-SNAPSHOT'
   }

Usage:

val t = textable {
    row {
        cell("Number")
        cell("Alphabet")
    }
    (0..25).forEach {
        row {
            cell("${it+1}")
            cell("${'A'+it}")
        }
    }
}

println(t)
